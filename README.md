# Projeto Integrador II - Produtiva

---

## 1. Levantamento de Requisitos Funcionais

- *[RF01]:* O sistema deve permitir o cadastro do usuário (fornecendo nome, e-mail e senha);
- *[RF02]:* O sistema deve permitir o login do usuário com e-mail e senha a uma conta existente;
- *[RF03]:* O sistema deve permitir ao usuário sair da conta (importante);
- *[RF04]:* O sistema deve permitir que apenas usuários e autenticados possam interagir com o sistema;
- *[RF05]:* O sistema deve permitir o gerenciamento de projetos (essencial), porém apenas o gerente pode fazê-lo;
- *[RF06]:* O sistema deve permitir o gerenciamento de tarefas (essencial), porém apenas o gerente pode fazê-lo;
- *[RF07]:* O sistema deve permitir ao gerente a definição de estimativas do tempo que será gasto em cada tarefa;
- *[RF08]:* O sistema deve permitir ao usuário registrar o tempo gasto ao trabalhar em uma tarefa e descreve o que foi feito em cada período (apontamento);
- *[RF09]:* O sistema deve permitir ao usuário visualizar relatórios de produtividade (baseando-se na comparação entre o tempo estimado e gasto em tarefas);
- *[RF10]:* O sistema deve permitir a categorização de tarefas com base em sua urgência e importância, classificando-as visualmente (com cores) (importante).
